# DrinksKings

DrinksKings é um projeto desenvolvido para demonstrar conceitos de desenvolvimento web moderno, gerenciamento de dados e autenticação de usuários. A aplicação simula uma plataforma digital voltada para o universo de bebidas, oferecendo uma experiência organizada e interativa para os usuários.

📖 Sobre o Projeto

O sistema foi estruturado para  funcionalidades encontradas em plataformas reais, incluindo autenticação de usuários, manipulação de dados e interface dinâmica.

🚀 Tecnologias Utilizadas

HTML5
CSS3
JavaScript
